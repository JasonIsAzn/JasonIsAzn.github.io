Billy Discord Bot

Billy Discord Bot is a personal Discord Bot written in Python. When playing League of Legends, information is critical to gaining an advantage on your opponent. This bot's main function is to provide user with information about League of Legends whether it's simple profile information to live in-game statistics. This project first started with using web scraping to obtain data, but throughout the project I realized this form of data gather was slow. While web scraping is useful I believe I need something else. This was when I first learned about RIOT API, which is an interface that allow me to directly use RIOT GAMES database with some limitations. I continued to learn by using REDDIT API, replit databases and overall improving my coding skill This discord bot also provide other function such as obtaining the latest news, memes, anime, and any addition functions that members of the discord suggest.
